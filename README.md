# Port Scanner (Python)

A simple TCP port scanner built using Python socket programming.

## Features
- Scans TCP ports from 1 to 65535
- Resolves domain name to IP address
- Displays open ports
- ASCII banner using pyfiglet
- Handles keyboard interruption gracefully

## Requirements
- Python 3.x
- pyfiglet

## Installation
```bash
pip install pyfiglet
